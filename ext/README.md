# External Skills — Compétences Externes Assimilées

Ce répertoire contient les skills externes intégrés dans ecosystem-1 via un processus de citoyennisation.

## Politique d'Assimilation

### Critères d'Éligibilité

1. **Licence Compatible**
   - MIT ✓
   - Apache 2.0 ✓
   - BSD ✓
   - GPL (avec copyleft) ✗

2. **Qualité du Code**
   - Tests présents
   - Documentation suffisante
   - Maintenance active

3. **Pertinence**
   - Alignement avec ecosystem-1
   - Réutilisable par ≥1 domaine

### Processus de Validation

```
1. Évaluation Licence
   ↓
2. Analyse Qualité Code
   ↓
3. Mapping vers Native Skills
   ↓
4. Identification Gaps
   ↓
5. Intégration Submodule
   ↓
6. Documentation ASSIMILATION.md
   ↓
7. Review L2/L3
   ↓
8. Merge & Publication
```
